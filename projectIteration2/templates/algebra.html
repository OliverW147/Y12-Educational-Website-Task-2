Maths - Algebra

Algebra is a branch of mathematics that deals with symbols and the rules for manipulating those symbols. These symbols (often represented by letters) stand in for numbers in equations and expressions, allowing us to generalize and solve problems.

Key Concepts in Algebra

Below are some foundational concepts in algebra, with explanations, examples, and interactive questions:

1. Variables and Expressions

A variable is a symbol, usually a letter, used to represent an unknown or changing value. For example, in the expression 2x + 3, x is a variable.

Example:

If x = 5, then 2x + 3 becomes 2(5) + 3 = 10 + 3 = 13.

Question: What is the value of 2x + 3 when x = 10? Enter your answer below:

Multiply 2 by 10 and then add 3.

2. Linear Equations

A linear equation is an equation where the highest exponent of the variable is 1. These equations often look like ax + b = c.

Example: Solve 2x + 3 = 11.

Solution:

  • Subtract 3 from both sides: 2x = 8
  • Divide both sides by 2: x = 4

Question: If 2x + 3 = 11, what is the value of x? Enter your answer below:

Try isolating x by subtracting and dividing.

3. Quadratic Equations

A quadratic equation is an equation where the highest exponent of the variable is 2. The standard form is ax² + bx + c = 0.

Example: Solve x² - 5x + 6 = 0 using factorization.

Solution:

  • Factor the quadratic: (x - 2)(x - 3) = 0
  • Set each factor to zero: x - 2 = 0 or x - 3 = 0
  • So, x = 2 or x = 3

Question: What are the roots of the equation x² - 5x + 6 = 0? Enter one root below:

Think about what two numbers multiply to 6 and add to -5.

4. Polynomials

A polynomial is a mathematical expression involving a sum of powers in one or more variables multiplied by coefficients. For example, 3x² + 2x - 1 is a polynomial.

Example: Add 3x² + 4x + 5 and 2x² - 3x + 1.

Solution:

  • Add the like terms: (3x² + 2x²) + (4x - 3x) + (5 + 1) = 5x² + x + 6

Question: What is the sum of 3x² + 4x + 5 and 2x² - 3x + 1? Enter your answer below:

Combine like terms: add the coefficients of the same powers.

5. Solving Word Problems

Word problems in algebra involve translating real-world situations into mathematical equations that can be solved.

Example: A number increased by 7 is equal to 12. What is the number?

Solution:

  • Let the number be x. The equation becomes x + 7 = 12.
  • Solve for x: x = 12 - 7 = 5.

Question: What number plus 7 equals 12? Enter your answer below:

Subtract 7 from 12.
